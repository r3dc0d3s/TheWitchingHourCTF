# Witching Hour CTF Challenges

Website: [Cyberz Club](https://cyberz.club/)

This repository contains a collection of CTF challenges designed, authored, and supervised by the Witching Hour Cybersecurity Club. Challenges cover multiple categories, including:

- **Crypto** – classic crypto problems, RSA, LCGs, bitflipping, and more  
- **Pwn** – memory exploitation, binaries, network analyzers  
- **Reverse Engineering (Rev)** – binaries, Rust challenges, Makefiles  
- **OSINT** – investigative and information gathering challenges  
- **Forensics** – packet capture and encrypted file analysis  
- **Web** – web exploitation and vulnerabilities  
- **Miscellaneous** – unique, experimental, or multi-category challenges

## Repository Structure

Each folder contains:

- `challenges/` – the problem files and Docker setups  
- `solutions/` – reference solutions and scripts
